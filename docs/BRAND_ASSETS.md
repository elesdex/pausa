# Pausa brand assets

These files are the canonical ring-free Pausa mark introduced on July 17, 2026. Use the SVG source for editing and the PNG exports for video, case-study, or promotional layouts.

## App icon

- Editable SVG: `public/brand/pausa-app-icon.svg`
- 1024px PNG: `public/brand/pausa-app-icon-1024.png`
- PWA exports: `public/pausa-icon-v2-192.png` and `public/pausa-icon-v2-512.png`
- iPhone touch icon: `public/pausa-apple-touch-icon-v2.png`

The app icon uses a cream rounded square, a forest-green circle, and the cream Pausa bars. It intentionally has no dotted or decorative outer ring.

## Standalone mark

- Editable SVG: `public/brand/pausa-mark.svg`
- Transparent 1024px PNG: `public/brand/pausa-mark-1024.png`

Use the standalone mark over light, calm backgrounds. Preserve clear space around the green circle and do not add an outer ring.
